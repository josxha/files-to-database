Demo: Speichere eine Liste von Dateien in der Datenbank ab.


1. Beim Aufruf der Seite wurde die Datenbank "MyDB" erstellt. Die unten angezeigten Bilder (1,2,3) wurden von der Webseite geladen.

2. Beim Klick auf den Button werden dre Bilder als BLOB Objekte hineingespeichert (muss beim installieren der App aufgerufen werden).

3. Nun sollten sich drei Bilder in der Datenbanktabelle befinden (Datenbank muss wahrscheinlich neu geladen werden, damit sie angezeigt werden).
